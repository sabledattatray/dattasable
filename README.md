# Dattasable.com

Welcome to the official repository for **Dattasable.com** — a modern, responsive, and creative personal blog and portfolio website designed and developed by [Datta Sable](https://dattasable.com).

## 📌 About the Project

Dattasable.com is a personal blog and web design portfolio created to share:
- Web development tutorials
- Digital design concepts
- Blogging tips
- Creative projects and resources

The project features:
- Responsive boxed layout
- Left-side drawer navigation
- Hero section with image, heading, and button
- Recent posts grid layout
- Custom footer and header with social icons and search bar
- Clean modern UI with dividers and styled sections

## 📦 Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript (for drawer menu & icons)**
- **Ionicons CDN**
