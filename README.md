# Local Development

## Clone Repository

```bash
git clone https://github.com/sabledattatray/dattasable.git
```

## Install Dependencies

```bash
npm install
```

## Configure Environment

```bash
cp .env.example .env
```

## Database Initialization

```bash
npx prisma db push
npx prisma db seed
```

## Start Development Server

```bash
npm run dev
```
