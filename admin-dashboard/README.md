# 📊 Enterprise Admin Dashboard (Classic WP Experience)

**A high-performance Management Infrastructure for Data-Driven Platforms.**

This dashboard is a sophisticated React-based administration portal designed to provide a "Classic WordPress" interface within a modern, high-speed ecosystem. It serves as the central command center for the **Datta Sable** analytics network, managing content, user data, and system configurations.

---

## 🎨 Architectural Identity

- **Classic WP Theme**: A customized "Obsidian Dark" theme that replicates the productivity of the WordPress admin panel while utilizing modern React performance.
- **ClassicWpLayout**: A dedicated layout engine designed for complex navigation and multi-pane management.
- **Analytics Integration**: Deeply integrated with the **SDR-9 Engine** for real-time monitoring of data sessions.

---

## 🛠️ Technical Stack

- **Framework**: React 18+ (Vite)
- **UI Components**: Material UI (MUI) & Custom Styled Components
- **State Management**: Redux Toolkit / React Context
- **Design System**: Atomic Design with a focus on high-fidelity "Obsidian" aesthetics.

---

## 🚀 Key Modules

1. **Analytical Control Center**: Real-time visualization of system metrics.
2. **Content Hub**: Streamlined management of blog posts, portfolio items, and metadata.
3. **User Management**: Secure authentication and preference tracking.
4. **Settings Engine**: Global configuration for the connected BI platforms.
